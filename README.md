# My Scalatra Web App #

## Build & Run ##

```sh
$ cd scalatra-circe-example
$ sbt
> run
```

If `browse` doesn't launch your browser, manually open [http://localhost:8080/](http://localhost:8080/) in your browser.

# Heroku #

## Deploy to Heroku ##

```sh
$ cd scalatra-circe-example
$ heroku create
$ git push heroku master
```
